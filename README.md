# API Gympass

## Descrição

A API Gympass é uma aplicação para gerenciar check-ins em academias. Ela também permite que os usuários consultem a lista de academias disponíveis com base em sua localização.

## Tecnologias

- Node.js v20
- Docker
- Docker Compose
- TypeScript
- Prisma
- PostgreSQL
- Fastify
- Vitest para testes

## Requisitos

- Node v20.17+
- Docker e Docker Compose
